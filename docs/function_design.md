```python
# 故事数据结构
class Story:
    id: str              # UUID
    user_id: str         # 创建者ID
    title: str           # 故事标题
    content: str          # 故事正文
    category: str         # 分类
    target_age: str       # 目标年龄
    status: str           # 状态: draft/published
    word_count: int       # 字数
    created_at: datetime  # 创建时间
    updated_at: datetime  # 更新时间
```

### 5.4 语音合成参数

```python
# 语音合成参数
class SynthesizeParams:
    story_id: str           # 故事ID
    voice_type: str         # 声音类型: female/male/child/custom
    voice_model_id: str      # 自定义声音模型ID (可选)
    speed: float             # 语速: 0.5-2.0, 默认1.0
    pitch: float             # 音调: 0.5-2.0, 默认1.0
    volume: float            # 音量: 0.0-1.0, 默认1.0
    bg_music_id: str         # 背景音乐ID (可选)
    bg_music_volume: float   # 背景音乐音量: 0.0-1.0, 默认0.3
```

```
  ↓
循环查询任务状态
  ↓
任务完成?
  ├─ 是 → 获取音频URL

  │        → 结束
  │
  └─ 否 → 显示进度
```

```
  ↓
```

## 7. AI工具链模块详细设计

### 7.2 核心处理流程
```
故事文本
    ↓
┌────────────────────────────────────────┐
│           AI工具链处理                   │
├────────────────────────────────────────┤
│                                        │
│  1. 故事分析                           │
│     ↓                                  │

│     ↓                                  │

│     ↓                                  │
│  5. 音频混音                           │
│     ↓                                  │
│  6. 视频合成 (可选)                   │
│     ↓                                  │
│  7. 最终输出                           │
│                                        │
└────────────────────────────────────────┘
    ↓
完整作品
```

### 7.3 子任务定义

| 任务类型 | 描述 | 输入 | 输出 |
|---------|------|------|------|
| story_analyze | 故事分析 | 故事文本 | 分析结果JSON |

### 7.4 任务调度策略

```python
# 任务依赖关系
```

## 11. 错误处理设计

### 11.1 错误码定义

| 错误码 | 描述 | 处理方式 |
|-------|------|---------|
| 1001 | 用户名已存在 | 提示用户更换用户名 |
| 1002 | 邮箱已注册 | 提示用户登录或找回密码 |
| 1003 | 密码错误 | 提示用户检查密码 |
| 1004 | 账号不存在 | 提示用户注册 |
| 1005 | Token过期 | 引导用户重新登录 |
| 2001 | 故事不存在 | 提示用户检查故事ID |
| 2002 | 无权限访问故事 | 提示用户联系所有者 |
| 3001 | 语音合成失败 | 提示重试或联系客服 |
| 3002 | 音频样本无效 | 提示重新上传 |
| 3003 | 声音模型训练失败 | 提示重试 |
| 4001 | 作品不存在 | 提示用户检查作品ID |
| 4002 | 导出失败 | 提示重试 |
| 5001 | AI服务调用失败 | 自动重试或提示稍后重试 |
| 5002 | 任务超时 | 提示查看任务状态 |
| 9001 | 服务器内部错误 | 提示联系客服 |
| 9002 | 数据库错误 | 提示稍后重试 |

### 11.2 错误展示规范
- 用户可见错误：友好提示，不展示技术细节
- 日志记录：错误详情写入日志
- 错误恢复：提供明确的恢复操作指引
